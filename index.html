What is PumpVC?

Your trading terminal,
now with a voice.

PumpVC is a Chrome extension that attaches a voice & text room to every Solana token — so you can coordinate, share alpha, and react in real time with other degens.

Before PumpVC, memecoin traders were scattered across random Telegram groups, missing calls as price moved. There was no way to know who else was watching the same chart at the same time.

PumpVC changes that. Install the extension, open any supported trading terminal, and a live voice channel automatically appears for that token's contract address. Just click to join — and start talking.

No setup. No servers to configure. No separate apps. Just voice, right where you trade.

Whitepaper

The PumpVC Protocol.

A technical and economic overview of PumpVC, the voice layer for Solana memecoin trading.

Contents
1. Overview 2. Technical Architecture 3. Room System 4. $PumpVC Token 5. Tokenomics 6. Security 7. Vision

📄 1. Overview

PumpVC is a Chrome browser extension that provides real-time voice and text communication infrastructure for Solana memecoin traders. It operates as a decentralised social layer on top of existing trading terminals including pump.fun, DexScreener, Axiom, GMGN, and BullX.

The core premise: traders watching the same chart should be able to talk to each other instantly, without leaving their terminal. PumpVC solves the coordination problem that has historically forced traders into fragmented Telegram groups, Discord servers, and private DMs.

Every token on Solana has a unique contract address (CA). PumpVC uses this CA as a room key — all users viewing the same token's chart are joined into the same voice and chat room, regardless of which supported platform they're on.

⚙️ 2. Technical Architecture

PumpVC is built on three interconnected layers:

  • Chrome Extension (MV3): A Manifest V3 extension with content scripts injected into supported trading terminals, a popup UI for the voice/chat interface, an offscreen document managing live audio streams, and a MAIN-world bridge script for Phantom wallet access.
  • Signalling Server: A Node.js/Express/WebSocket server handling WebRTC offer/answer exchange, ICE candidate relaying, room management, user presence, and chat message persistence. Deployed on Railway for low-latency global access.
  • CA Resolution Engine: A multi-layer fallback system that resolves pair addresses (used by Axiom/DexScreener) to canonical token mint addresses. Resolution chain: DexScreener pairs API → on-chain Solana RPC decoding (Raydium AMM V4 + Orca Whirlpool byte offsets) → pump.fun API → DexScreener tokens API.

Audio is transmitted peer-to-peer via WebRTC after initial signalling. The server never handles audio data — only the handshake. This ensures minimal latency and maximum privacy.

🚪 3. Room System

Rooms are identified by a composite key: {token_ca}:{room_type} where room_type is either public or holders.

  • Public Rooms: Open to anyone — no wallet required. Any trader navigating to a supported terminal page for a given token joins the same public room. Zero friction.
  • Holders-Only Rooms: Gated by on-chain token balance verification. Users connect their Phantom wallet; the server queries the Solana RPC to confirm they hold ≥1 token of the CA in question. Access is re-verified on reconnect.
  • Room Persistence: Rooms exist as long as at least one participant is connected. Chat history is stored server-side and loaded on join. Voice connections are ephemeral (peer-to-peer).

🪙 4. $PumpVC Token

$PumpVC is the native utility token of the PumpVC ecosystem, deployed on Solana. It functions as a gating mechanism for premium features — not a speculative instrument. Holding $PumpVC unlocks functionality within the extension itself.

The token model follows a "hold-to-access" design pattern: as long as a user's wallet holds the required amount of $PumpVC, they retain access to premium features. There is no staking, no lock-up period, and no vesting — just hold and use.

Current unlock threshold: 1,000,000 $PumpVC for full Premium access. Thresholds are adjustable by governance as the supply dynamics evolve.

🔐 6. Security

  • No private key access: PumpVC only reads your public wallet address and token balance via Phantom's read-only API. We never request signing authority and cannot initiate transactions.
  • Peer-to-peer audio: Voice data never passes through PumpVC servers. WebRTC audio is end-to-end between participants.
  • On-chain verification: Holder status is verified live against the Solana RPC — no self-reporting, no screenshots, no trust required.
  • Open source: The extension's content scripts and background logic are open source and auditable. You can verify exactly what code runs in your browser.
  • MV3 compliance: Built on Manifest V3, Chrome's latest security-hardened extension architecture. No background scripts with persistent execution.

🔭 7. Vision

The long-term vision for PumpVC is to become the voice layer of on-chain trading — not just for Solana memecoins, but for every token on every chain. Just as Discord became the voice layer for gaming, PumpVC aims to become the voice layer for DeFi.

The memecoin market on Solana is the perfect starting point: high-velocity, community-driven, and acutely sensitive to social coordination. Traders who can communicate in real time have a genuine information advantage. PumpVC democratises that advantage.

Future expansion includes: Ethereum and Base chain support, a standalone trading terminal with PumpVC natively embedded, institutional API access for platforms wanting to integrate voice natively, and a reputation system built around on-chain trading history and community participation.

PumpVC is not just a tool — it's infrastructure for the new wave of on-chain social trading.

FAQ

Common questions.

Is PumpVC free to use? +
Yes — completely free. Install the Chrome extension, open any supported trading terminal, and you're in. No signup, no wallet required for public rooms. Holders-only rooms just need a Phantom connection to verify your balance.
Does PumpVC access my wallet or funds? +
No. PumpVC only reads your public wallet address and token balance. We never request transaction signing permissions and cannot initiate transfers of any kind. Your funds are completely safe.
What is $PumpVC and do I need it? +
$PumpVC is the extension's utility token. You don't need it for basic use — all voice rooms and chat are free. Holding $PumpVC unlocks premium cosmetics like GIF sending, profile borders, custom fonts, and the holder badge.
How does voice quality compare to Discord? +
PumpVC uses WebRTC peer-to-peer audio, the same technology Discord uses. Voice data travels directly between participants — our servers only handle the initial connection handshake, keeping latency minimal.
Why doesn't it work on [site X]? +
PumpVC currently supports pump.fun, DexScreener, Axiom, GMGN, and BullX. We're actively adding new platforms. If your preferred terminal isn't supported, let us know and we'll prioritise it.
How does pair address resolution work? +
Some platforms (Axiom, DexScreener) display Raydium pool addresses in URLs rather than token mints. PumpVC automatically resolves these via a multi-layer fallback: DexScreener API → Solana RPC on-chain decode → pump.fun API. This ensures everyone sees the same room.
Can I use PumpVC on mobile? +
Not yet — Chrome extensions don't run on mobile. A dedicated iOS and Android companion app is on the roadmap for Q4 2025, with push notifications for whale entries and room activity.
When are the holder cosmetics shipping? +
GIF chat and the PumpVC holder badge are live now for $PumpVC holders. Animated profile borders, custom fonts/colors, and exclusive emote packs are actively in development and shipping in Q3 2025.
